# Coca-Cola Stock Price Analysis

## Objective
To analyze historical Coca-Cola stock data and identify trends and risk.

## Tools Used
Python, Pandas, Matplotlib

## Steps Performed
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Trend analysis using moving averages
- Volatility and return analysis

## Conclusion
The analysis shows that Coca-Cola stock has a strong long-term growth trend with relatively low volatility.
